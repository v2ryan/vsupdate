# VS Code 2025年7月更新 (版本 1.103)

## 📅 發佈日期
2025年7月

---

## 🌟 主要功能摘要

| 類別 | 功能名稱 | 說明 |
|------|----------|------|
| **模型** | GPT-5 支援 | 新增 GPT-5 模型 |
| **Chat** | Chat 檢查點 | 保存同恢復 Chat 會話狀態 |
| **Chat** | 工具分組 | 將相關工具分組顯示 |
| **Git** | Git Worktree 支援 | 完整支援 Git 工作樹 |
| **Agent** | 多回應建議 | Agent 提供多個解決方案選擇 |
| **終端** | 終端 IntelliSense | 增強終端智能完成 |

---

## 📝 詳細功能列表

### 模型支援

| 模型 | 描述 | 用途 |
|------|------|------|
| GPT-5 | OpenAI 最新模型 | 更強大嘅推理同代碼生成 |
| Claude Sonnet 4 | Anthropic 模型 | 替代選擇 |
| Gemini 2.5 Pro | Google 模型 | 多模態支援 |

### Chat 檢查點

| 功能 | 描述 |
|------|------|
| 保存檢查點 | 隨時保存當前 Chat 狀態 |
| 恢復檢查點 | 返回到之前保存嘅狀態 |
| 分支對話 | 從任意點開始新嘅對話分支 |
| 工作區狀態 | 同時保存相關嘅工作區變更 |

### 工具分組

| 功能 | 描述 | 設定 |
|------|------|------|
| 分組顯示 | 相關工具組合顯示 | - |
| 自訂分組 | 定義自己嘅工具分組 | 工具集設定 |
| 快速訪問 | 更易找到需要嘅工具 | - |

### Git Worktree 支援

| 功能 | 描述 |
|------|------|
| 完整支援 | 喺源碼控制視圖中顯示 Worktree |
| 新增 Worktree | 輕鬆創建新嘅工作樹 |
| 切換 Worktree | 快速喺工作樹之間切換 |
| 刪除 Worktree | 管理工作樹生命週期 |

### Agent 改進

| 功能 | 描述 |
|------|------|
| 多回應建議 | 提供多個解決方案選項 |
| 更好嘅錯誤恢復 | 自動從錯誤中恢復 |
| 改進上下文理解 | 更準確嘅代碼分析 |
| 終端監控 | 更好嘅命令執行追蹤 |

### 終端 IntelliSense

| 功能 | 描述 | 設定 |
|------|------|------|
| 命令完成 | 智能命令建議 | `terminal.integrated.suggest.enabled` |
| 參數提示 | 命令參數說明 | - |
| 歷史建議 | 基於歷史嘅建議 | - |

### NES 改進

| 功能 | 描述 |
|------|------|
| 更快觸發 | 改進觸發速度 |
| 更準確建議 | 提高建議質量 |
| 語法高亮 | 建議中嘅完整語法高亮 |

### 編輯器體驗

| 功能 | 描述 | 設定 |
|------|------|------|
| 改進搜索 | 更快嘅搜索性能 | - |
| 資源管理器改進 | 更好嘅檔案導航 | - |
| 主題更新 | 視覺改進 | - |

### 擴充功能開發

| 功能 | 描述 |
|------|------|
| MCP 穩定 API | MCP 相關 API 穩定 |
| 工具 API 改進 | 增強工具開發體驗 |
| 語言伺服器改進 | LSP 增強 |

---

## 🔧 重要設定

```json
{
  "terminal.integrated.suggest.enabled": true,
  "git.worktrees.enabled": true,
  "github.copilot.chat.checkpoints.enabled": true
}
```
