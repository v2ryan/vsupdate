# VS Code 2025年11月更新 (版本 1.107)

## 📅 發佈日期
2025年11月

## 🌟 主要功能摘要

| 類別 | 功能名稱 | 說明 |
|------|----------|------|
| **Agent** | Agent HQ | 集中管理 Agent 會話同任務 |
| **Agent** | 背景 Agents | Agent 可以喺背景運行 |
| **MCP** | MCP 2025-11-25 規範 | 支援最新 MCP 規範 |
| **TypeScript** | TypeScript 7.0 支援 | 支援 TypeScript 7.0 |
| **Chat** | 更智能嘅模型 | 改進嘅模型推薦 |
| **編輯器** | 性能優化 | 持續嘅性能改進 |
